# Day 10 – Attach Elastic IP to EC2

## Task

The Nautilus DevOps team is continuing their AWS migration by configuring networking components for their infrastructure.

For this task, you need to attach an existing Elastic IP to an EC2 instance.

## Requirements

| Parameter | Value |
|-----------|------|
| Instance Name | nautilus-ec2 |
| Elastic IP Name | nautilus-ec2-eip |
| Region | us-east-1 |

Attach the Elastic IP **nautilus-ec2-eip** to the EC2 instance **nautilus-ec2**.

Ensure that the Elastic IP is successfully associated with the instance.
