# Day 04 -- Enable S3 Versioning

## Task

The Nautilus DevOps team wants to ensure data protection and recovery
for their S3 storage.

For this task, you need to enable versioning on an existing S3 bucket.

## Requirements

  Parameter     Value
  ------------- -------------------
  Bucket Name   devops-s3-3446
  Feature       Enable Versioning
  Region        us-east-1

Enable versioning for the given S3 bucket to protect data from
accidental deletion or overwrites.
