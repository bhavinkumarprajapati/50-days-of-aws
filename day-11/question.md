# Day 11 – Attach Elastic Network Interface to EC2

## Task

The xFusion DevOps team is continuing their AWS migration by configuring advanced networking components for their infrastructure.

For this task, you need to attach an existing Elastic Network Interface (ENI) to an EC2 instance.

## Requirements

| Parameter | Value |
|-----------|-------|
| Instance Name | xfusion-ec2 |
| ENI Name | xfusion-eni |
| Region | us-east-1 |

Attach the network interface **xfusion-eni** to the EC2 instance **xfusion-ec2**.

Ensure that the ENI status is **attached** before submitting the task.

> **Note:** Make sure instance initialisation has been completed before submitting.
