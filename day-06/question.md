# Day 06 – Create EC2 Instance

## Task

The Nautilus DevOps team is continuing their AWS migration by provisioning compute resources.

For this task, create an **EC2 instance** with the following configuration.

## Requirements

| Parameter | Value |
|-----------|------|
| Instance Name | xfusion-ec2 |
| AMI | Amazon Linux |
| Instance Type | t2.micro |
| Key Pair | xfusion-kp (RSA) |
| Security Group | Default |
| Region | us-east-1 |

Ensure that a new RSA key pair named **xfusion-kp** is created and used while launching the instance.
