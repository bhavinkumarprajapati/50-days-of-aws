# 50 Days of AWS Challenge (KodeKloud)

This repository documents my journey of completing the **100 Days of AWS Challenge** on KodeKloud.

The goal of this challenge is to strengthen my AWS and DevOps skills through daily hands-on tasks.

## Challenge Structure

Each day contains:

- QUESTION.md → Task description
- SOLUTION.md → Implementation and explanation
- assets → screenshots or diagrams
