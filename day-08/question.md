# Day 08 – Enable EC2 Stop Protection

## Task

As part of the AWS migration, the Nautilus DevOps team needs to ensure that critical instances are protected from accidental shutdown.

For this task, enable **stop protection** on an existing EC2 instance.

## Requirements

| Parameter | Value |
|-----------|------|
| Instance Name | devops-ec2 |
| Feature | Stop Protection |
| Region | us-east-1 |

Ensure that the EC2 instance cannot be accidentally stopped.
