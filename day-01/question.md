# Day 01 – Create AWS Key Pair

## Task

The Nautilus DevOps team is planning to migrate part of their infrastructure to the AWS cloud. To ensure a smooth transition, they have decided to perform the migration gradually by breaking large tasks into smaller units.

For this task, create an AWS key pair with the following configuration.

## Requirements

- Key Pair Name: **xfusion-kp**
- Key Pair Type: **RSA**

Use the provided AWS credentials to log in to the AWS console and complete the task.
