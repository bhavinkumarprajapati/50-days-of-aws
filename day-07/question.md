# Day 06 – Modify EC2 Instance Type

## Task

The Nautilus DevOps team is optimizing their AWS infrastructure by identifying underutilized resources and adjusting configurations accordingly.

For this task, you need to modify an existing EC2 instance.

## Requirements

1. Change the instance type of **xfusion-ec2** from:
   - t2.micro → t2.nano

2. Ensure that the instance status checks are completed before making changes.

3. After modification, the instance **must be in running state**.

4. Region: us-east-1
