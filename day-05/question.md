# Day 05 – Create EBS Volume

## Task

The Nautilus DevOps team is continuing their AWS cloud migration by provisioning storage resources required for their infrastructure.

For this task, create an **EBS (Elastic Block Store) volume** with the following configuration.

## Requirements

| Parameter | Value |
|-----------|------|
| Volume Name | xfusion-volume |
| Volume Type | gp3 |
| Size | 2 GiB |
| Region | us-east-1 |

The volume should be created in the appropriate availability zone within the selected region.
